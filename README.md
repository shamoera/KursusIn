# KursusIn
Tugas Besar Mata Kuliah Algoritma Pemrograman 2

## Deskripsi
KursusIn merupakan sistem pendaftaran kursus online terpadu yang digunakan untuk mengelola data peserta kursus.

## Fitur
- Menambahkan data peserta
- Menampilkan data peserta
- Searching data peserta
- Sorting data peserta
- Statistik peserta aktif

## Bahasa Pemrograman
- Golang
